<< task2 문서 설명 >>


1. 긴 문장 text data 수집
    yeonhapnews_webcrawling.py 파일은 긴 문장 data로, 연합 뉴스 기사 50개를 크롤링하여 csv파일로 저장하는 코드이다.
    데이터에서 불필요한 구성요소인 '<[^>]*>'를 제거한 후 저장하였다. '\n' 또한 모두 '.'으로 처리하였다. ('.'은 형태소 분석에 영향을 미치지 않는다.)

    save_t.csv는 위의 코드를 실행시켜 저장된 csv파일이다.

    body_data.tst는 csv파일에서 body부분 텍스트만 추출하여 저장한 txt 파일이다. 


2. 정규화, 토큰화
    task2_data_preprocessing.py파일은 데이터를 토큰화하기 전에 정규화를 진행하는 코드가 작성되어 있는 파일이다. 
    데이터 토큰화를 진행하는 코드를 시행중인 파일로, kiwipiepy(키위형태소 분석기)를 사용하였다. 
    현재 명사를 추출하는 코드가 작성되어 있다.

    앞에서 수집한 데이터에서 명사만 뽑아 추출한 후,
    불필요한 명사들을 제거하여 상위 30개의 명사를 나열한 후 저장하였다.


3. word cloud 시각화
    wordcloud.png파일은 앞에서 저장한 상위 30개 단어로 word cloud를 수행한 이미지 파일이다.


4. 과제를 수행하며 시행착오를 겪은 사항
    task2_trial_and_error 문서 확인.


5. 전체 코드 설명
    task2_code_explain파일은 text data 수집, 데이터 정규화, 토큰화, 시각화를 진행하는 코드를
    모두 과정별로 설명해둔 파일이다.
